# 校园共享厨房小程序

## 项目简介

校园共享厨房小程序是一个基于 uni-app 开发的跨平台应用，旨在为校园用户提供厨房预订、食材购买等服务，丰富校园生活。

### 主要功能

- **用户认证**：支持账号密码登录
- **厨房预订**：浏览热门厨房、查看详情、预约使用
- **食材购买**：浏览食材、加入购物车、提交订单
- **个人中心**：查看个人信息、订单和预订记录

## 技术栈

### 前端
- **框架**：uni-app
- **语言**：Vue.js
- **样式**：CSS/SCSS
- **UI**：自定义组件
- **网络请求**：uni.request

### 后端
- **框架**：Flask
- **语言**：Python
- **数据库**：MySQL
- **认证**：JWT
- **CORS**：Flask-CORS

## 快速开始

### 前端开发

1. **安装依赖**
   ```bash
   npm install
   ```

2. **运行开发服务器**
   ```bash
   npm run dev:mp-weixin
   ```

3. **在微信开发者工具中导入**
   - 打开微信开发者工具
   - 选择「导入项目」
   - 选择 `unpackage/dist/dev/mp-weixin` 目录
   - 填写AppID（可使用测试号）
   - 点击「导入」

### 后端开发

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **配置数据库**
   - 在 `app/config/database.py` 中修改数据库连接信息
   - 默认配置：`mysql+pymysql://root:111111@localhost:3306/campus_share_kitchen`

3. **启动后端服务**
   ```bash
   python app.py
   ```

4. **服务地址**
   - 默认运行在 `http://127.0.0.1:5000`

## API文档

### 用户相关

- **登录**：`POST /api/user/login`
- **注册**：`POST /api/user/register`
- **获取用户信息**：`GET /api/user/info`

### 厨房相关

- **获取厨房列表**：`GET /api/kitchen/list`
- **获取厨房详情**：`GET /api/kitchen/detail`
- **预约厨房**：`POST /api/reserve/add`
- **获取预约列表**：`GET /api/reserve/list`

### 食材相关

- **获取食材列表**：`GET /api/food/list`
- **提交订单**：`POST /api/order/add`
- **获取订单列表**：`GET /api/order/list`
